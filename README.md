- **Balance**. I balance pragmatism with efficiency to keep things moving forward by mixing the context-driven testing and the holistic testing approaches with modern tools and automation. 

- **Asynchronous collaboration**. Asynchronous collaboration helps teams minimize face-to-face call times while also maximizing focused work time and  preventing knowledge silos.
